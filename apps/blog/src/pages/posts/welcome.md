# Welcome to Minions Profile

We're excited to introduce **Minions Profile** — Your source of truth — verified bio claims, portfolio items, and rate cards.

## What's Included

- **TypeScript SDK**: `npm install @minions-profile/sdk`
- **Python SDK**: `pip install minions-profile`
- **CLI**: `npm install -g @minions-profile/cli`
