# Minions Profile

Your source of truth — verified bio claims, portfolio items, and rate cards

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-profile/sdk`) — Core library for Node.js
- **Python SDK** (`minions-profile`) — Python bindings
- **CLI Tool** (`@minions-profile/cli`) — Command-line interface
- **Web App** — Interactive playground at [profile.minions.wtf](https://profile.minions.wtf)
