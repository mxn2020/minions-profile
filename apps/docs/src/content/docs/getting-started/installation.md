---
title: Installation
description: Install Minions Profile packages
---

## TypeScript / Node.js

```bash
npm install @minions-profile/sdk minions-sdk
```

## Python

```bash
pip install minions-profile
```

## CLI

```bash
npm install -g @minions-profile/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
